WikiGeo

WikiGeo is a class project that I developed with my teammates for CS411: Software Engineering in Spring 2016 at Boston University.



The basic idea behind this application is to help people learn more about the places near them. For example, when you travel to a new city, you might get curious about the places that you are visiting. What kind of monuments are there? Did any historical events happen around the area? When you have these kinds of questions, WikiGeo is here to help!

This web application use geolocation and find where the user currently is, then it will display an interactive map showing markers of historical events or monuments nearby. By clicking on the marker, a detailed description and pictures of that specific event/monument will pop up, and the user can then learn more about that location.

We also include the feature that allows users to create their own markers of the locations they know. They can create a discription and upload photos of the place. This marker will then become public and be shared with other users in the community.


How to Use this App?

Below is a quick guide.


Search Page

When you log in to the app, here is the main page you will see. By default, the search area will be your current location. You can also change your search area by moving the little black circle on the map. The circle will move to place you clicked, and it updates the searching area.

Whenever you are lost on the map, you can always go back to your current location by simply clicking "Show my location" button, and it will bring you back to your current location.


Search Result

After you decide your search area, click on the "WikiGeo" button. A number of markers will pop up on your map. When you scroll down, there will also be a reference table.


Learn more about the search result

When you click on a marker, the name of the marker will show up. You can then click on that name, and it will take you that marker page.

On the marker page, you can find a picture of that monument/event. There is also a description of the place you just found.

Adding your own markers

To add a marker, click on the "Add Marker" button and it will take you to the Add Marker Page.

Choose the location on the map, and then fill in the information about this place.

Finally, your marker is added successfully! Now, other users can also see this marker and learn about this place.


Technical Stuff

This web application is developed using MEAN stack (MongoDB, Express, Angular, Node.js). We use Google Maps API to acquire user's geolocation and cross reference the geolocation with the data pulled from wikipedia API. The result is then displayed on an interactive Google Map.
